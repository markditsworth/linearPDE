Verify second-order convergence in the eigenvalues.

In [6]:
lambdas = zeros(4);
idx = 1;
for m = [100 200 400 800] # iterate through difference M sizes, doubling each time
    D = diff1(m);         # calculate D matrix for given m size
    dx = L/(m+1);         # calculate dx for given m
    x = dx*0.5:dx:L;      # get x sequence
    C = diagm(0 => c(x)); # get C matrix
    A = -D'*C*D;          # calculate A
    l,U = eigen(A);       # find eigenvalues
    lambdas[idx] = l[end];# save smallest magnitude eigenval
    idx += 1;
end
print("Show the decrease factor from a doubling of resolution\n")
println(abs(lambdas[2]-lambdas[1]) /abs(lambdas[3]-lambdas[2]))
println(abs(lambdas[3]-lambdas[2]) /abs(lambdas[4]-lambdas[3]))

A doubling of resolution results in a quadruple decrease in difference, thus the eigenvalues are second-order convergent.
